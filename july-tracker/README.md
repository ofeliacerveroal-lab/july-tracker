# July Tracker 💚

App de seguimiento de salud para July y su clienta.

## Qué incluye

- 📸 Registro de fotos de comidas (desayuno, comida, cena, tentempié)
- 💊 Control de tensión arterial (7 días, mañana y noche)
- ⚖️ Seguimiento semanal de peso, cintura y caderas
- 📋 Guía completa con las indicaciones de July
- 👩‍⚕️ Vista especial para July (resumen, galería de fotos, progreso)

## Cómo desplegar en Vercel

1. Sube este repositorio a GitHub
2. Ve a [vercel.com](https://vercel.com) y entra con tu cuenta de GitHub
3. Haz clic en "Add New Project"
4. Selecciona este repositorio
5. Vercel detecta automáticamente que es React — dale a Deploy
6. ¡Listo! Comparte la URL con July

## Uso local

```bash
npm install
npm start
```
